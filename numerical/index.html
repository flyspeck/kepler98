Inequality Testing

The proofs of parts of the Kepler conjecture rely on several dozen inequalities. These inequalities have been checked numerically. They have also been proved rigorously. This page describes resources that are available for checking the inequalities numerically.

There is an java applet that tests various inequalities that arise. This applet is inefficient for large scale testing, and we have used C++ for the serious tests described below.

Each of the inequalities has been given an identification number. There are several lines of C++ source code for each inequality.

  • Part III, inequalities for tetrahedra
  • Part III, inequalities for octahedra
  • Part III, inequalities for flat quarters
  • Part III, inequalities for pure quad clusters
  • Part III, inequalities for flat quarters
  • Part IV, all inequalities

To check each inequality (F>0), we mimimized F by numerical methods using the program CFSQP which is available, free of charge, to non-profit organizations from the Institute for Systems Research at the University of Maryland.
